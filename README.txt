This is my implementation of pintos project.
The specification of Pintos is available here: https://web.stanford.edu/class/cs140/projects/pintos/pintos.html
My implementation is tested within QEMU simulator.
Results: Project 1 27/27 passed. Project 2:76/76 passed.
		 Project 3 109/109 passed. Project4: 144/155 passed.
You need to modify some scripts of Pintos to make my implementation run on 
your machine. Regarding this, please refer to:
